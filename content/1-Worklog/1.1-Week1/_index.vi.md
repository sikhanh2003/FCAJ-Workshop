### Thành tựu Tuần 1:

* Kết nối với các thành viên FCJ và ghi chú quy định thực tập.

* Có hiểu biết cơ bản về Cloud Computing.

* Học các nguyên tắc cơ bản của DevOps Engineering.

* Khám phá khái niệm CI/CD qua tài liệu.

* Hiểu về AWS cơ bản và các dịch vụ cốt lõi:
  * Compute
  * Storage
  * Networking
  * Authentication
  * Server

* Tạo và cấu hình tài khoản AWS Free Tier.

* Làm quen với AWS Management Console để truy cập dịch vụ.

* Cài đặt và cấu hình AWS CLI, bao gồm:
  * Access Key
  * Secret Key
  * Default Region

* Sử dụng AWS CLI cho các hoạt động cơ bản:
  * Kiểm tra thông tin tài khoản
  * Liệt kê các vùng
  * Xem dịch vụ EC2
  * Quản lý key pairs
  * Kiểm tra thông tin về các dịch vụ đang chạy

* Hoàn thành 5 nhiệm vụ để nhận 200 Free Tier credits.

* Học các kiến thức cơ bản về EC2:
  * Loại instance
  * AMI
  * Key pair
  * Security Group

* Khám phá tổng quan hạ tầng AWS.

* Hiểu về cách sử dụng Elastic IP.

* Có kỹ năng quản lý tài nguyên qua Console và CLI song song.
