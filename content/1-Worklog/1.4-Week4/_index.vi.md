* Tạo ra ý tưởng cho Project 1 qua nghiên cứu.

* Học các kiến thức cơ bản về cơ sở dữ liệu với Amazon RDS.

* Thực hành tạo instance EC2 và RDS.

* Triển khai một ứng dụng tích hợp EC2 và RDS.

* Khám phá các tính năng của AWS Support.

* Thiết lập môi trường phát triển với AWS Toolkit trong VS Code.

* Thực hành tạo một trường hợp hỗ trợ.

* Cấu hình VS Code với AWS Toolkit cho phát triển.

* Khám phá AWS SageMaker cho workload AI trong Project 1.

* Điều tra hosting AI models trên EC2 trong Project 1.

* Học về Next.js cho Project 1.

* Khám phá Animate React cho animations trong Project 1.

* Có hiểu biết về AWS Lambda với API Gateway.

* Xây dựng một blog công nghệ cá nhân sử dụng Next.js và Animate React.

* Tạo hàm Lambda cho việc tạo thumbnail tự động trên các tải lên S3.

* Xây dựng một API sử dụng API Gateway tích hợp với các hàm Lambda.

* Nâng cao kỹ năng trong kiến trúc serverless, cơ sở dữ liệu và tích hợp AI.
